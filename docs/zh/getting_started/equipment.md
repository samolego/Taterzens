# 设置NPC的装备

---


## 武装它们！

输入以下指令进入该NPC的装备编辑模式：
```
/npc edit equipment
```
进入装备编辑模式后可执行以下操作：

* 手持装备/武器对着NPC右键即可让NPC手持或者穿戴。
* 对着NPC左键即可交换NPC主副手上的物品。
* 手持装备对着NPC按住Shift键+右键可让NPC手持装备。
* 空手对着NPC然后按下**Shift键+右键**可清空NPC身上所有的装备。

如果想让NPC的装备在死亡时掉落而不是*直接清除*，请输入以下指令：
```
/npc edit equipment allowEquipmentDrops true
```
